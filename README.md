# Fusarium_evolution
make_sc_og_species_gene_table.py
Function
Generate an Orthogroup × Species gene presence table from
Orthogroups.tsv and Orthogroups_SingleCopyOrthologues.txt produced by OrthoFinder.
Usage
python3 make_sc_og_species_gene_table.py \
  -og_tsv orthofinder_results_dir/Orthogroups/Orthogroups.tsv \
  -sc_file orthofinder_results_dir/Orthogroups/Orthogroups_SingleCopyOrthologues.txt \
  --out_wide sc_og_wide.tsv \
  --out_long sc_og_long.tsv \
  --expect_species_n N

Arguments
-og_tsv
Path to the Orthogroups.tsv file generated by OrthoFinder.
-sc_file
Path to Orthogroups_SingleCopyOrthologues.txt.
--out_wide
Output filename for the wide-format table (Orthogroups × Species).
(Optional: filename can be customized.)
--out_long
Output filename for the long-format table.
(Optional: filename can be customized.)
--expect_species_n
Expected number of species in the dataset (integer).
